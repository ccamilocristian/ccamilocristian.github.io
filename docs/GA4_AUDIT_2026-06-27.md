### Flujo de medición actual

```
Página carga
  → consent-defaults (analytics_storage: denied)
  → CookieYes banner
  → GTM (siempre carga el container)
  → Usuario acepta analytics → gtag consent update → tags GA4 en GTM deberían disparar
  → Usuario acepta ads → adsbygoogle.js carga
```

**Riesgo:** si GTM no tiene consent settings publicados, GA4 puede **no registrar** usuarios que aceptan el banner, o registrar de forma inconsistente.
