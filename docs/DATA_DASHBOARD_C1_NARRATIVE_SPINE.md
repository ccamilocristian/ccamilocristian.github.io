# Data Dashboard - C1 Narrative Spine Rewrite

Task: Content Strategy C1 (RED)

## Objective

Unify the site narrative so all sections communicate one operating story:

Data Analysis -> Machine Learning -> Business Intelligence -> Economics-informed decisions -> measurable impact.

## Narrative spine definition

1. Data Analysis: convert noisy data into trustworthy signals.
2. Machine Learning: estimate outcomes and compare alternatives.
3. Business Intelligence: operationalize insights for recurring decisions.
4. Economics: frame tradeoffs, constraints, incentives, and risk.
5. Vision Lab: emerging capability that follows the same decision discipline.
